# Sayı İşlemleri Fonksiyonları

Bu repository, JavaScript ile yazılmış üç farklı sayı işlemleri fonksiyonunu içermektedir: arkadaş sayıları bulma, mükemmel sayıları bulma ve asal sayıları bulma.

## Dosya Yapısı

- `js/main.js` dizini altındaki dosyada üç farklı JavaScript Method bulunmaktadır:
  - `friendNumbers`: Arkadaş sayıları bulma fonksiyonunu içerir.
  - `perfectNumbers`: 1000'e kadar olan tüm mükemmel sayıları bulma fonksiyonunu içerir.
  - `primeNumbers`: 1000'e kadar olan tüm asal sayıları bulma fonksiyonunu içerir.
