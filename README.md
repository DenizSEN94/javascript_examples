# Sayı İşlemleri Fonksiyonları

Bu repository, JavaScript ile yazılmış üç farklı sayı işlemleri fonksiyonunu içerir: arkadaş sayıları bulma, mükemmel sayıları bulma ve asal sayıları bulma.

## Fonksiyonlar

### Arkadaş Sayılar Bulma

`main.js` dosyasında yer alan fonksiyon, parametre olarak girilen iki sayının arkadaş sayılar olup olmadığını bulur.

Kullanım:
```javascript
friendNumbers({ sy1: 220, sy2: 284 });
